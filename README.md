# Информационные технологии и программирование (1-й семестр)

Учебный репозиторий для студентов 1 курса направления **«Прикладная математика и информатика»**. Курс посвящен глубокому изучению основ программирования на языке C# и платформе .NET.

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

## 🚀 О курсе
Курс рассчитан на **3 семестра**.  
**Форма контроля:**
*   1-й семестр — экзамен.
*   2-й и 3-й семестры — зачет с оценкой.

**Условия допуска к экзамену/зачету:**
1. Отсутствие прогулов (прогулы отрабатываются конспектом и доп. заданиями).
2. Своевременная сдача всех лабораторных работ.

| **[Лекция 1](Лекции/Лекция_1.pptx)** | Введение в C# | История языка, компоненты ПО, компиляция (JIT/Native), система типов. |

## 🧪 Лабораторные и практические работы

### Лабораторные работы
1. **ЛР №1:** Линейные программы и класс `Math`.
2. **ЛР №2:** Разветвляющиеся и циклические алгоритмы.
3. **ЛР №3:** Работа с массивами данных и строковыми объектами.
4. **ЛР №4:** Отладка программ и оценка временной сложности.

### Практические занятия
1. Консольный ввод/вывод и форматирование строк (`$""`).
2. Операторы выбора и тернарный оператор.
3. Написание методов, работа с `out` и кортежами.
4. Оценка сложности алгоритмов на примерах циклов.

## 🏆 Большой проект: «Анализ текста»
Итоговая работа семестра, объединяющая все знания:
*   Очистка текста от мусора (регулярные выражения/StringBuilder).
*   Разбиение на предложения и слова.
*   Фильтрация стоп-слов через `HashSet`.
*   Подсчет частоты слов через `Dictionary`.
*   Вывод ТОП-10 популярных слов (сортировка).
