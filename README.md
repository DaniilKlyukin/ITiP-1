# Информационные технологии и программирование (1 семестр)

Данный репозиторий содержит учебные материалы для студентов 1 курса направления **«Прикладная математика и информатика»**. Курс посвящен изучению основ программирования на языке C# и платформе .NET.

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

## 🏆 Итоговый проект: «Анализ текста»
Практический проект по обработке больших текстовых данных на примере книги о **Гарри Поттере**.
*   **Цель:** Превратить «хаос букв» в полезную статистику.
*   **Технологии:** `List`, `Dictionary`, `HashSet`, `StringBuilder`, `Array.Sort`.
*   [Ссылка на подробности в лекции 7](Лекции/ИТиП%201%20лек%207.pptx)
